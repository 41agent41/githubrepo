# Ferguson Consulting Services Website

Official website for Ferguson Consulting Services (frgsn.au)

## Overview

A modern, responsive business website built with HTML5, CSS3, and vanilla JavaScript. Features a clean, professional design optimized for all devices.

## Pages

- **Home** (`index.html`) - Landing page with hero section and key features
- **About** (`about.html`) - Company information, mission, vision, and values
- **Services** (`services.html`) - Detailed service offerings
- **Contact** (`contact.html`) - Contact information and inquiry form

## Features

- Fully responsive design (mobile-first approach)
- Modern CSS with CSS Grid and Flexbox
- Smooth scrolling and animations
- Mobile navigation menu
- Contact form with validation
- Accessible (ARIA labels, semantic HTML)
- SEO optimized

## Technology Stack

- HTML5
- CSS3 (modern features: Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (ES6+)

## Getting Started

Simply open `index.html` in a web browser to view the website locally. No build process or dependencies required.

## Deployment

This static website can be deployed to any web hosting service:
- GitHub Pages
- Netlify
- Vercel
- AWS S3 + CloudFront
- Traditional web hosting

## Customization

### Colors

Edit CSS variables in `styles.css` to change the color scheme:

```css
:root {
    --primary-color: #2563eb;
    --primary-dark: #1e40af;
    /* ... more variables */
}
```

### Content

Update the HTML files directly to modify text, add images, or change layout.

### Contact Information

Update the email and phone number in the footer and contact page with your actual contact details.
